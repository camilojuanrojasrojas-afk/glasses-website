# 🕶️ Glasses Website Platform

## 🚀 Overview
Interactive web platform built with HTML5 that integrates 10 calculation tools into a single, user-friendly interface. The system also includes SQL-based data management for efficient data storage and processing.

---

## 💡 Features
- 10 integrated calculation tools  
- Age, weight, and math calculations  
- Simple and intuitive interface  
- SQL data storage  
- Fast and responsive performance  

---

## 🛠️ Tech Stack
- HTML5  
- CSS3  
- JavaScript  
- SQL
